# T4: ハイスコア画面 + セーブのインポート/エクスポート (games/records.js)

## 概要
全ゲームのハイスコアを一覧表示する「Records」画面と、セーブデータの JSON エクスポート/インポート機能。

## 前提(データ仕様)
core.js はスコアを localStorage キー `scores_v1` に**配列**形式で保存する:

```json
[
  { "gameId": "breakout", "score": 120, "date": "2026-07-05" },
  { "gameId": "reflex", "score": 88, "date": "2026-07-05" }
]
```

この形式を前提に実装すること。

## 要件
- バニラ JS。外部ライブラリ・CDN 禁止

## 受入基準
1. メニューから「Records」画面を開くと、全ゲームのハイスコア一覧が表示される
2. 「Export」ボタンでセーブデータを JSON ファイルとしてダウンロードできる
3. 「Import」でJSON ファイルを選択すると、内容を検証したうえで反映され、一覧表示が更新される
4. 不正な JSON(構文エラー・形式違反)はエラーメッセージを表示して拒否する
5. インポートした内容がページリロード後も保持される
