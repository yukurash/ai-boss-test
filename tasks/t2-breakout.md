# T2: ブロック崩し (games/breakout.js)

## 概要
メインゲーム。canvas 製のブロック崩し。レベル 3 面、ライフ制、スコアは T1 の `Arcade` API で保存する。gameId は `breakout`。

## 要件
- バニラ JS + canvas。外部ライブラリ・CDN 禁止
- T1 の `core.js` が提供する `Arcade.submitScore('breakout', score)` を使用する(自前の保存機構を作らない)

## 受入基準
1. メニューから起動し、canvas にゲーム画面が描画される
2. パドルが ← → キーで動く
3. パドルがマウス移動でも動く
4. ボールが壁・パドル・ブロックで反射する(パドルの当たり位置で反射角が変わること)
5. ブロックをすべて壊すと次レベルへ進む(全 3 レベル、ブロック配置がレベルごとに異なる)
6. ボールを落とすとライフが減る(初期ライフ 3)。0 でゲームオーバー
7. スコアが画面に表示される(ブロック破壊で加点、レベルクリアでボーナス)
8. ゲームオーバー時と全レベルクリア時に `Arcade.submitScore('breakout', score)` が呼ばれる
9. ゲームオーバー/クリア画面からリトライとメニュー復帰ができる
10. 一時停止と再開ができる(Esc キーまたはボタン)

## 手動確認チェックリスト(完了報告に結果を含めること)
- [ ] レベル 1 から通しでプレイし、レベル遷移またはゲームオーバーまでを実際に確認した
- [ ] リロード後にハイスコアが残っていることを確認した

## 簡易テスト(必須)
`tests/breakout.test.html` を作成し、ブラウザで開くと純粋ロジック(反射計算、レベルデータの整合性、スコア計算など)のアサーションが最低 5 件実行され、結果が画面に表示されるようにすること。完了報告に全件の合否を含めること。
